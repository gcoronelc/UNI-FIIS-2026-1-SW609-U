Nested Loop
Como buscar cada nombre de una lista corta en una agenda — funciona bien si la lista es pequeña.
Hash Join
Como organizar contactos por inicial en cajones — eficiente, pero requiere espacio para armarlos.
Merge Join
Como fusionar dos listas ya ordenadas — muy rápido, pero solo si ambas listas ya están en orden.
